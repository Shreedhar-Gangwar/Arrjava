How publishing works
  1. Compose the publication below. Write paragraphs normally; a blank line starts a new paragraph. Begin a line with ## to make it a sub-heading.
  2. Click Preview to see it exactly as readers will.
  3. Click Download publications.js — the file will contain all existing publications plus this one, newest first.
  4. Replace the publications.js file on your hosting with the downloaded file. The publication is live immediately.

A standard disclaimer ("This note is for general information only…") is appended automatically.
